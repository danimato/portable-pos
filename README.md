# Portable POS
# [Visit the Wiki!](https://github.com/danimato/portable-pos/wiki)
# [Visit the page too](https://danimato.github.io/portable-pos/)

The Portable POS (Point of Sale) system is a lightweight and versatile solution designed for businesses that require a mobile and efficient way to manage sales transactions. Built with portability in mind, this system can be easily deployed on various devices, making it ideal for small businesses, pop-up shops, and market vendors.
